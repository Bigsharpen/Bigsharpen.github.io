函数（方法）

  • 一种工具——>封装性
  • 只需要知道怎么使用
  • 我们学习的所有函数都是某个对象的方法

    对象（字符串，日期，Math，数组，正则）

  • 工具包：把相似的工具（功能）放在一起，便于使用
  • 包含两种东西：函数，属性
  • 万物皆对象
  • 整个js都是面向对象的

    对象以及实例化

  • 对象是抽象，实例是具体
  • 手机是抽象，不同型号的手机产品是实例
  • 我们只能使用实例，不能使用抽象化的东西
  • 要使用对象要先实例化：var 实例 = new 对象（）；

    面向对象编程的步骤：

  1. 思考需要哪几个对象
  2. 分析属性和方法，先把空壳（接口）写出来
  3. 实现工具：写代码封装成函数和对象
  4. 使用工具
